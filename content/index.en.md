# Discover Airflow 3 Workshop

Welcome! 🚀

This workshop is designed to get you familiar with some of the biggest new features in Apache Airflow 3.X. You'll learn about the new UI, Assets (the evolution of datasets), human-in-the-loop workflows, backfills, DAG versioning, and event-driven scheduling.

## Workshop Overview

In this hands-on workshop, you'll build an automated personalized newsletter system using Airflow 3's latest features. The workshop includes:

- **ETL Pipeline**: Retrieve data from APIs, format it, and create newsletter templates
- **Personalization Pipeline**: Generate personalized newsletters based on user preferences
- **Advanced Features**: Human-in-the-loop approval, backfills, and GenAI integration

![Demo architecture diagram](/static/img/etl_genai_newsletter_architecture_diagram_bedrock.png)

## Learning Objectives

By the end of this workshop, you will:

- Navigate the new Airflow 3 React-based UI
- Implement asset-oriented scheduling for data pipelines
- Add human-in-the-loop operators for manual intervention
- Execute backfills for historical data processing
- Track DAG changes with versioning
- Understand event-driven scheduling patterns

## Prerequisites

This workshop requires basic Python and Airflow knowledge. All setup is handled through the preparation sections below.
