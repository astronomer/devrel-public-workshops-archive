# Prepare Required Sandbox Environments

This section covers the sandbox environments needed for the workshop, focusing on the Astronomer platform setup that provides the core Airflow 3 experience.

## Sandbox Overview

### Primary Environment: Astronomer Astro
- **Cloud-based Airflow platform**: No local installation required
- **Airflow 3.0 ready**: Latest features pre-configured
- **Free trial access**: No credit card required for workshop
- **Managed infrastructure**: Focus on learning, not setup

### Optional Environment: AWS Services
- **For Exercise 6 only**: Advanced GenAI and event-driven features
- **Personal AWS account**: Required for SQS and Bedrock access
- **Cost-effective**: Minimal charges for workshop completion
