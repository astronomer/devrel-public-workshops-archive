# Prepare AWS Environment

This workshop can be run in different AWS environments depending on your access and requirements. Choose the appropriate setup method below.

## Environment Options

### Core Workshop (Modules 1-5)
- **No AWS account required**
- Uses Astronomer's managed platform
- Complete Airflow 3 feature exploration
- Recommended for most participants

### Advanced Workshop (Module 6)
- **AWS account required**
- Includes GenAI and event-driven features
- Amazon SQS and Bedrock integration
- Optional advanced capabilities
